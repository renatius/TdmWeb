Searching for a Pattern
Previous  Top  Next


To search a text for a given pattern:

·Type the pattern in the Pattern Edit Box and set any relevant compile options as described in Compiling a Pattern  
·Type the text you want to search for your pattern in the Text To Search Edit Box  
·Set any appropriate Search Options  
·Press the Search Button  
 
 
 
Note that in addition to display the full match, the Matches list contains also all capture groups plus the string to the left of the match and the string following it.
